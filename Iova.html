    Iowa AEA: Iowa Library Portal

    Iowa Library project allows easy access to school media resources acquired from a broad array of vendors. One of the prominent features of the product is flexible federated search solution that allows to keep up-to-date list of media resources and distribute it across according to the policies defined. It allows flexible collaboration between students and teachers using notes/quotes and discussion boards. It supports various bibliographic and cataloging formats which makes vendors metadata integration swift and effortless.To facilitate school student management system integration Iowa Library project was developed with Schools Interoperability Framework in mind. Iowa Library portal has its own reader applications that allow students and teachers to read any content (most popular media formats are supported) on any device (mobile and web version available). It makes easy access to educational information whenever you are. You can listen to audio books, watch video content, read your books accompanied by supplemental services such as translations/text-to-speech/bookmarks/notes/quotes and overall collaboration. For teachers it allows flexible and easy interface to prepare and distribute learning material among their students and keep track of the studentsâ€™ learning advancement. Vendors, that integrate their resources, have capacity (via API) to receive timely feedback about resources usage. That allows better control and insights into material distributed.
